This task focuses on performing manual exploratory testing on a live web application and documenting real-time issues in a structured and professional bug report format. 

The selected application for testing:
https://practice.automationtesting.in/

The Objective was to identify genuine usability,UI,and validation issues and document them using industry-standard bug reporting practices.

ISSUES IDENTIFIED
A total of 5 genuine issues were found, including:
- Country dropdown not visible in registration form.
- Phone number format inconsistency
- Scrollbar visibility issue in web table
- Image misalignment in mobile view
- Navigation dropdown misalignment

TOOLS USED
- Google Chrome
- Chrome Devtools
- GitHub

This task demonstrates practical manual testing skills, UI defect identication, and professional bug documentation.
